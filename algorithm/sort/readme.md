- 冒泡排序 Bubble Sort
    1. 排序
    2. 交换
    3. 相邻
    4. 有序区域
    5. 稳定排序


    交换排序(位置, 从小到大， 从大到小)
- 最小值和最大值, 不能传整个数组, 使用es6展开运算符
    Math.min(...array)
    Math.max(...array)

- 如果一次交换都没有发生
    通过检测本轮是否有交换  如果没有 待排序区域已有序  可以退出  isSorted 布尔值

- 如果后半有有序区域
    基于上次交换的位置的下标优化

- 冒泡排序的时间复杂度
    3个维度？
    最好时间复杂度      最坏时间复杂度      平均时间复杂度
    O(n) 本来就是有序   O(n^2) 倒序        O(n^2)


- 插入排序的关键点
    1. 当前元素前面的那个序列是有序的。有序区间 
    2. 所有在当前元素前面的数都不大于它，所有在当前元素后面的数都不小于它
    3. 一个个往后移动，让待排序元素插入新的位置
