- 电商类移动端首页  采用瀑布流效果
    错落有致  美观
- 特定组件封装

- 编程里碰到难得问题怎么解决
    - 细分法
        1. 2列式的弹性布局
            flex + jc space-between
        2. vue 数据绑定
            allGoods{
                pic:'',
                title:'',
                height:''
            }
            leftGoods
            rightGoods
        3. 瀑布流会把哪一列最短 就把数据放到那一列
